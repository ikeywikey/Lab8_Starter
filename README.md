## Graceful Degradation and Service Workers

Service workers let a site keep working even when there's no internet. When a user goes offline, the service worker steps in and serves saved copies of the pages and data instead of showing an error. The site might not have the latest content, but it still works. Checking whether the browser supports service workers before using them means the site won't break for users who don't have that support either.
